# Gmail Cleanup Script

This Google Apps Script automatically moves unwanted emails to the Trash.

To use it:

1. Open [Google Apps Script](https://script.google.com/).
2. Paste in the code.
3. Click **Run ▶️ myFunction**.
4. Approve Gmail permissions.

You can change the search query in the script to target different senders or labels.
Use small batch sizes to avoid timeouts.

Once moved to the Trash, Gmail will permanently delete messages after 30 days.
